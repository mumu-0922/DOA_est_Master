# 第3章 基于坐标注意力网络的低信噪比 DOA 估计方法

## 3.1 方法总体框架

描述“样本协方差矩阵 -> 四通道输入 -> CA-DOA-Net -> 空间谱 -> 峰值搜索”的完整流程。

## 3.2 四通道协方差矩阵输入特征构造

写实部、虚部、相位正弦、相位余弦四类特征及其作用。

## 3.3 坐标注意力机制

解释坐标注意力如何沿两个方向编码，并迁移到协方差矩阵行、列方向阵元相关结构增强。

## 3.4 CA-DOA-Net 网络结构

写网络主干、注意力模块位置、输出维度。

## 3.5 空间谱输出与峰值搜索

写网格化空间谱、峰值检测、NMS/最小间隔约束。

## 3.6 损失函数设计

主写空间谱损失与谱稀疏约束。分离约束若涉及非可导峰值搜索，需要谨慎表述。

## 3.7 本章小结

总结本文方法如何针对低 SNR 下协方差特征受噪声污染的问题做增强。
