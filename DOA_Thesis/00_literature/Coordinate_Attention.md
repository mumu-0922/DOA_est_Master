# Coordinate Attention for Efficient Mobile Network Design

## 论文作用

坐标注意力将通道注意力分解为两个一维方向编码，在增强通道表达的同时保留方向位置信息。迁移到 DOA 估计时，可将协方差矩阵视为阵元间相关结构图，沿行、列方向增强阵元相关特征。

## 与本文方法的对应关系

| Coordinate Attention 原概念 | 本文迁移解释 |
| --- | --- |
| 特征图的水平/垂直方向编码 | 协方差矩阵行/列方向的阵元相关结构编码 |
| 保留位置信息的通道注意力 | 保留阵元索引关系的特征重标定 |
| 轻量注意力模块 | 不显著增加网络复杂度的谱估计增强模块 |

## 写作注意

- 不能写成 CA 原本就是为 DOA 设计。
- 应写成“受坐标注意力启发，将其方向编码能力迁移到协方差矩阵特征建模中”。
- 需要结合本文输入矩阵的物理意义解释行、列方向。
