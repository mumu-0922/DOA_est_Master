# 术语统一

| 中文术语 | 英文/符号 | 建议写法 |
| --- | --- | --- |
| 波达方向 | DOA | 首次出现写“波达方向(Direction of Arrival, DOA)” |
| 均匀线阵 | ULA | 本文默认阵列结构 |
| 样本协方差矩阵 | SCM, R_hat | 不要与理想协方差矩阵混用 |
| 信噪比 | SNR | 单位 dB |
| 快拍数 | snapshots, T | 全文统一用 T 表示 |
| 信源数 | K | 不要与角度网格数量混淆 |
| 阵元数 | M | 默认 M = 8 |
| 空间谱 | spatial spectrum | 网络输出形式 |
| 坐标注意力 | Coordinate Attention, CA | 注意区分通道注意力 SE 和 CBAM |
| 成功率 | Success Rate, SR | 定义为在容差内成功匹配全部/指定信源的比例，需在第 4 章明确 |
| 均方根误差 | RMSE | 单位 deg |

## 避免表述

- 避免“首次提出”。
- 避免“彻底解决低信噪比 DOA 问题”。
- 避免把仿真实验写成实测验证。
- 如果某个损失项不可导或通过后处理得到，不写成严格端到端物理约束。
